# Portfolio Heritiana Andrianantenaina — version UI/UX professionnelle

Version sombre premium adaptée à la nouvelle photo de profil : contraste clair, accents cuivre, navigation fixe discrète, projets cliquables et contact direct.

## Lancer le projet

```bash
npm install
npm run dev
```

Adresse locale habituelle : http://localhost:5173/

## Build production

```bash
npm run build
npm run preview
```

## Points importants

- Navbar fixe, visible pendant le scroll, avec fond sombre transparent pour ne pas gêner la lecture.
- Accueil plus direct : profil, chiffres, compétences, projets, contact.
- Projets affichés en images avec titres courts ; les détails complets s’ouvrent au clic.
- Contact simplifié : email du visiteur, message, bouton envoyer.
- Envoi via FormSubmit vers : heritiana.andrianantenaina24@gmail.com

Au premier message réel, FormSubmit peut demander une validation de l’adresse email de réception. C’est normal.
