# Installation rapide Windows

1. Dézippez le dossier.
2. Ouvrez Git Bash ou le terminal dans le dossier du projet.
3. Lancez :

```bash
cd portfolio-vue-heritiana-fullstack
npm install
npm run dev
```

4. Ouvrez l’adresse affichée, généralement :

```text
http://localhost:5173/
```

Si `vite` n’est pas reconnu, cela signifie que `npm install` n’a pas terminé correctement. Supprimez `node_modules`, relancez `npm install`, puis `npm run dev`.
