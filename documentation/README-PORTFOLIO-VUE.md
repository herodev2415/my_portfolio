# Documentation — Portfolio Vue.js

## Pourquoi cette version est meilleure que le HTML simple

Cette version renforce ton positionnement de développeur Fullstack car le portfolio est lui-même une application frontend structurée : composants Vue, routing, données centralisées, assets organisés et configuration de déploiement.

## Points importants pour les recruteurs

- Le portfolio montre que tu sais organiser un projet moderne.
- Chaque projet présente les fonctionnalités, les défis techniques et les solutions.
- Les technologies sont clairement associées à des réalisations concrètes.
- Le design reste clair, lisible et professionnel au lieu d’être surchargé.
